## 学习建议

1. 建议从基础概念开始，理解 Kafka 的核心设计理念
2. 然后按照安装部署指南搭建环境
3. 之后可以按照生产者、消费者的顺序深入学习
4. 在实践过程中遇到问题，可以参考存储原理和高可用架构章节
5. 最后是运维与开发相关的实用工具和最佳实践
